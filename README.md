# Breezart Home Assistant Integration

Кастомная интеграция для Home Assistant для систем вентиляции Breezart 550 Aqua.

## Возможности

- Подключение к Breezart по TCP (порт 1560)
- Чтение датчиков:
  - Текущая температура
  - Статус системы
  - Скорость вентилятора
  - Температура воды
  - Температура улицы
- Управление:
  - Целевая температура
  - Скорость вентилятора
  - Включение/выключение

## Установка

### Способ 1: HACS (рекомендуется)

1. В HACS → Интеграции → ⋯ → Пользовательские репозитории
2. Добавить: `https://github.com/egorhomeriga/breezart-ha`
3. Установить интеграцию
4. Перезапустить Home Assistant
5. Настроить через UI: Настройки → Интеграции → + → Breezart

### Способ 2: Ручная установка

1. Скопировать папку `custom_components/breezart` в `<config>/custom_components/`
2. Перезапустить Home Assistant
3. Настроить через UI

## Настройка

- **Хост**: IP-адрес Breezart (например, 192.168.1.121)
- **Порт**: 1560 (стандартный порт Breezart)
- **Пароль**: Пароль для доступа (например, 21579)

## Разработка

Интеграция основана на протоколе из `breezart-client` и `homebridge-breezart`.

### Структура

```
custom_components/breezart/
├── __init__.py          # Основная логика
├── config_flow.py       # UI настройки
├── coordinator.py       # TCP клиент и координатор
├── sensor.py           # Сенсоры
├── number.py           # Управление числами
├── switch.py           # Выключатели
├── const.py            # Константы
├── manifest.json       # Метаданные
└── translations/       # Переводы
```
